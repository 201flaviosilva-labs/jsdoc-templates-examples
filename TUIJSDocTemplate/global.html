    rockPaperScissor(player1, player2) → {number}
    Games/rockPaperScissor.js, line 26

    Lets play a game. Rock Paper Scissors.

    This function will check which is the winner of the game, based on the options:

    • "r" for rock;
    • "p" for paper;
    • "s" for scissor;

    And determines the winner:

    • 0 for draw
    • 1 for player 1
    • 2 for player 2
    Parameters:
    Name Type Description
    player1 string

    the player 1 option
    player2 string

    the player 2 option
    Returns:
    number
    Examples
    rockPaperScissor("p", "p"); // 0
    rockPaperScissor("r", "r"); // 0
    rockPaperScissor("s", "s"); // 0
    rockPaperScissor("p", "r"); // 1
    rockPaperScissor("s", "r"); // 2

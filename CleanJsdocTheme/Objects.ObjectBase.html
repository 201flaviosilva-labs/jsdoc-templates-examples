Objects. ObjectBase

The base class for a 2D objects. Probably you don't need to use this class directly. Use this class to extend your own objects.

Constructor

new ObjectBase(x, y, fillColoropt, strokeColoropt)

Parameters:
NameTypeAttributesDefaultDescription
xnumber

The horizontal position of this Object in the world.

ynumber

The vertical position of this Object in the world.

fillColornumber | string<optional>
0xffffff

The color the Object will be filled with, i.e. 0xff0000 for red.

strokeColornumber | string<optional>
0x000000

The color of the border of the Object, i.e. 0x00ff00 for green.

Source
  • 2DObjects/ObjectBase.js, line 1
Example
class MyObject extends ObjectBase {
	constructor(x, y, fillColor, strokeColor, ...args) {
		super(x, y, fillColor, strokeColor);
	}
}

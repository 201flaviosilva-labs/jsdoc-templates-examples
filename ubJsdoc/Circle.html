The Circle Shape is a 2D Object.

  
      const circ = new Circle(400, 300, 50, "#ff0000", 0x00ff00);

# new Circle (x:  number, y:  number, radiusopt:  number, fillColoropt:  number | string, strokeColoropt:  number)

Arguments:
  • x: number

    The horizontal position of this circle.

  • y: number

    The vertical position of this circle.

  • radius = 10: number

    The radius of this circle.

  • fillColor = 0xffffff: number| string

    The color the circle will be filled with, i.e. 0xff0000 for red.

  • strokeColor = 0x000000: number| string

    The color of the border of the circle, i.e. 0x00ff00 for green.

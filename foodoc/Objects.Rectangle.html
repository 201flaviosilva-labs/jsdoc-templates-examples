  1. Home

new Rectangle( x, y [, width [, height [, fillColor [, strokeColor ] ] ] ] )

Description

The Rectangle Shape is a 2D Object. Its possible to set a rounder corner radius.

Parameters
Name Type Attributes Default Description
x number

The horizontal position of this rectangle
y number

The vertical position of this rectangle
width number <optional>
 100

The width of the rectangle.
height number <optional>
 100

The height of the rectangle.
fillColor number | string <optional>
 0xffffff

The color the rectangle will be filled with, i.e. 0xff0000 for red.
strokeColor number | string <optional>
 0x000000

The color of the border of the rectangle, i.e. 0x00ff00 for green.
Examples
const rect = new Rectangle(400, 300, 75, 50, "#ff0000", 0x00ff00);

factorial(number) → number

The factorial of a non-negative​ number is computed as the product of all integers between 1 and the number itself

Examples 

factorial(0); // 0
factorial(1); // 1
factorial(1.2); // 0.23999999999999994
factorial(4); // 24
factorial(6); // 720
factorial(10); // 3628800
